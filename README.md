## Featured Startup Project: NeuChar / NeuCharBox

NeuChar is a modular software platform centered around NeuCharFramework (NCF), with a product and developer ecosystem that includes:

- a core application framework
- official package sources
- documentation tooling
- AI integration modules
- WeChat and social-platform management modules
- sample projects for onboarding and experimentation
